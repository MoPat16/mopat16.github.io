Programming for Social Science module

The creation of this website was to provide a portfolio of my code for the module's assessment 1. The module provided an introduction in computer programming for social scientists.

What was required?

For assessment 1 the criteria was to create an Agent Based Model following the instructions provided through the practicals. The code that I have provided includes a programme that creates a list of agents which interact with the environment. The agents also interact with each other by sharing information. The model also includes stopping conditions where the agents stop moving within the environment and also stop sharing information. Finally, the code also has a GUI and includes the downloading of data from the Internet to initiate further interaction between the agents.
